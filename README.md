# python_aws_chalice
RESTful endpoints for a polls app using AWS Chalice

- Install:

```
pip install chalice

pipenv install chalice
```

- Create a project:

```
chalice new-project project_name
```

- Run local:
```
chalice local
```

- Deploy:

```
chalice deploy --profile profile_name
```

- Delete all:
```
chalice delete --profile profile_name
```
